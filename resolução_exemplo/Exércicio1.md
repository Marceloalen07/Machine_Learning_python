# 🧠 Classificação de Clientes com Machine Learning: Previsão de Compra

## 📌 Descrição

Neste exercício, você irá desenvolver um modelo de **Machine Learning** capaz de prever se um cliente irá comprar um produto com base em suas características.

A atividade simula um problema real de negócio, onde empresas utilizam dados para tomar decisões estratégicas.

---

## 🎯 Objetivo

Criar um modelo de **classificação supervisionada** utilizando Python e a biblioteca Scikit-learn para prever a variável **"comprou"**.

---

## 📊 Base de Dados

O conjunto de dados contém as seguintes variáveis:

* **idade** → idade do cliente
* **salario** → salário do cliente
* **comprou** → se o cliente comprou (1) ou não (0)

---

## 🧩 Etapas do Exercício

Você deverá:

1. Carregar o arquivo CSV no Google Colab
2. Visualizar os dados
3. Separar as variáveis:

   * X (idade, salario)
   * y (comprou)
4. Dividir os dados em treino e teste
5. Criar um modelo de **Regressão Logística**
6. Treinar o modelo
7. Fazer previsões
8. Avaliar o modelo utilizando:

   * Acurácia
   * Matriz de confusão

---

## 💡 Perguntas Teóricas

Responda:

1. Esse problema é de classificação ou regressão? Explique.
2. O aprendizado é supervisionado ou não supervisionado?
3. O que significa acurácia?
4. O que a matriz de confusão representa?
