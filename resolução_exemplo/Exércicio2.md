# 🏠 Previsão de Preço de Casas com Machine Learning

## 📌 Descrição

Neste projeto, você irá desenvolver um modelo de **Machine Learning** para prever o **preço de uma casa** com base em características como tamanho, número de quartos e idade do imóvel.

Este é um problema clássico de **regressão**, muito utilizado no mercado imobiliário e em análise de dados.

---

## 🎯 Objetivo

Criar um modelo de **regressão supervisionada** capaz de estimar o preço de uma casa.

---

## 📊 Base de Dados

O conjunto de dados contém as seguintes variáveis:

* **tamanho (m²)** → área da casa
* **quartos** → número de quartos
* **idade** → idade da casa (anos)
* **preco** → preço da casa

---

## 🧩 Etapas do Exercício

Você deverá:

1. Criar ou carregar a base de dados (CSV)
2. Visualizar os dados
3. Separar:

   * X (tamanho, quartos, idade)
   * y (preco)
4. Dividir os dados em treino e teste
5. Criar um modelo de **Regressão Linear**
6. Treinar o modelo
7. Fazer previsões
8. Avaliar o modelo utilizando:

   * MAE (Erro Absoluto Médio)
   * MSE (Erro Quadrático Médio)
   * R² (Coeficiente de Determinação)

---

## 💡 Perguntas Teóricas

1. Esse problema é de classificação ou regressão?
2. Qual a diferença entre MAE e MSE?
3. O que significa o R²?
4. Um erro alto indica o quê no modelo?
